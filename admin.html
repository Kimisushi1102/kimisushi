CẤU HÌNH DÀNH CHO VPS

Gắn Tên Miền & SSL

Cách dễ nhất để gõ tên miền là vào thẳng web (không cần số :3000) và có ổ khóa xanh HTTPS.

CÔNG CỤ HỖ TRỢ TỰ ĐỘNG

Tôi đã chuẩn bị sẵn file deploy-helper.ps1 trong thư mục dự án. Bạn chỉ cần chạy nó bằng PowerShell để hệ thống tự động cài đặt mọi thứ!

1

Chuẩn bị Cloudflare

Bạn cần đăng ký tài khoản Cloudflare dashboard (Miễn phí) và trỏ Tên miền của mình về đây.

2

Tạo "Đường hầm" (Tunnel)

Vào menu Zero Trust -> Networks -> Tunnels. Nhấn Create a Tunnel, đặt tên là "Sakura-POS", sau đó nhấn Save.

3

Kích hoạt trên VPS (Chạy Lệnh)

Chọn Windows và copy lệnh cài đặt (Install Service) mà Cloudflare cung cấp, sau đó dán vào PowerShell (Admin) trên VPS của bạn.

cloudflared-windows-amd64.exe service install [TOKEN_CUA_BAN]
4

Cấu hình Tên Miền

Trong phần Public Hostname trên Cloudflare:

  • Public Hostname: Nhập tên miền của bạn (VD: sakurasushi.com)
  • Service Type: Chọn HTTP
  • URL: Nhập localhost:3000

Bây giờ bạn có thể gõ trực tiếp Tên miền là vào thẳng web với HTTPS chuyên nghiệp!
