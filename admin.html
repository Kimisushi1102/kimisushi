Cách đơn giản nhất để có HTTPS miễn phí với ổ khóa xanh

CÔNG CỤ HỖ TRỢ

Chạy file deploy-helper.ps1 trong thư mục dự án để tự động cài đặt!

1

Chuẩn bị Cloudflare

Đăng ký tài khoản Cloudflare và trỏ tên miền về đây.

2

Tạo Tunnel

Vào Zero Trust → Networks → Tunnels → Create Tunnel, đặt tên "KimiPOS".

3

Cài đặt trên VPS

Chạy lệnh cài đặt từ Cloudflare trong PowerShell (Admin).

cloudflared-windows-amd64.exe service install [TOKEN]
4

Cấu hình Public Hostname

  • Hostname: Tên miền của bạn (VD: kimisushi.de)
  • Service Type: HTTP
  • URL: localhost:3000

Hoàn tất! Giờ bạn có thể truy cập website qua tên miền với HTTPS!
